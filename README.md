#### 信发项目汇总
信息发布软件Android客户端相关的项目

#### 基础项目内容
* app:小信发项目（局域网）广告机端
* clientconnection:小信发项目（局域网）手机端
* h5xinfa:通过WebView+H5的形式加载下发文件的形式
* xinfafirstversion:最初信发版本，可通过设置坐标动态的显示信发文件
